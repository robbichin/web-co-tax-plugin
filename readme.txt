Allow users to toggle taxes on your WooCommerce store in your navigation, as a widget, with a shortcode and/or as a floating button.

== Description ==
Streamline tax control on your WooCommerce store - users will now be able to toggle taxes seamlessly via navigation, a widget, shortcode, or a handy floating button.

With only 30 lines of CSS code, this plugin will not interfere with your current theme, customizer or WooCommerce color scheme. You can also override colours and change the \'Display tax?\' text in each of the four locations.

This solution ensures a user-friendly interface while giving you complete control of your customers\' shopping experience.

== Installation ==
1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Under \'Settings\', click on \'Woo Customer Tax Toggle\'
4. Full instructions for each of the four modules are there under settings
